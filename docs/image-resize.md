# image-resize

## Synopsis

Resizes an image.

## Description

Uses ImageMagick to resize an image to specified dimensions.

## Signature

```powershell
image-resize
```

## Parameters

### -InputPath

The path to the input image file.

### -OutputPath

The path for the output image file.

### -Width

The desired width.
