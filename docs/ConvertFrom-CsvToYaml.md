# ConvertFrom-CsvToYaml

## Synopsis

Converts CSV file to YAML format.

## Description

Reads a CSV file and outputs its contents as YAML.

## Signature

```powershell
ConvertFrom-CsvToYaml
```

## Parameters

### -Path

The path to the CSV file to convert.
