# Add-DirectoryBookmark

## Synopsis

Creates a directory bookmark.

## Description

Saves the current directory as a named bookmark for quick access.

## Signature

```powershell
Add-DirectoryBookmark
```

## Parameters

### -Name

Name for the bookmark.

### -Path

Directory path to bookmark (defaults to current directory).
