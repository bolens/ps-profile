# gl

## Synopsis

Shows commit history.

## Description

Displays the commit log for the repository. Forwards all arguments to git log.

## Alias Information

**Alias for:** `Get-GitLog`

This is an alias that points to the `Get-GitLog` command. Use this alias as a shorthand for the full command name.
