# confirm

## Synopsis

Shows a confirmation prompt using gum.

## Description

Displays an interactive confirmation dialog using gum. Returns true if confirmed, false otherwise.

## Alias Information

**Alias for:** `Invoke-GumConfirm`

This is an alias that points to the `Invoke-GumConfirm` command. Use this alias as a shorthand for the full command name.
