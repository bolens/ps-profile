# Test-IsLinux

## Synopsis

Checks if the current platform is Linux.

## Description

Returns true if running on Linux, false otherwise.

## Signature

```powershell
Test-IsLinux
```

## Parameters

No parameters.

## Outputs

System.Boolean. True if running on Linux, false otherwise.
