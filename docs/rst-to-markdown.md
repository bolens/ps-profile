# rst-to-markdown

## Synopsis

Converts RST file to Markdown.

## Description

Uses pandoc to convert a reStructuredText (RST) file to Markdown format.

## Alias Information

This is an alias that points to the `ConvertFrom-RstToMarkdown` command. Use this alias as a shorthand for the full command name.
