# ConvertTo-Base64

## Synopsis

Encodes input to base64 format.

## Description

Converts file contents or string input to base64 encoded string.

## Signature

```powershell
ConvertTo-Base64
```

## Parameters

### -InputObject

The file path or string to encode.
