# latex-to-rst

## Synopsis

Converts LaTeX file to RST.

## Description

Uses pandoc to convert a LaTeX file to reStructuredText (RST) format.

## Alias Information

This is an alias that points to the `ConvertTo-RstFromLaTeX` command. Use this alias as a shorthand for the full command name.
