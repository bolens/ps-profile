# docx-to-html

## Synopsis

Converts DOCX file to HTML.

## Description

Uses pandoc to convert a Microsoft Word DOCX file to HTML format.

## Alias Information

This is an alias that points to the `ConvertTo-HtmlFromDocx` command. Use this alias as a shorthand for the full command name.
