# open

## Synopsis

Opens files or URLs using the system's default application.

## Description

Opens the specified file or URL using the appropriate system command.
        On Windows, uses Start-Process. On Linux/macOS, uses xdg-open or open.

## Signature

```powershell
open
```

No parameters.
