# navib

## Synopsis

Finds the best matching command from navi cheatsheets.

## Description

Searches navi cheatsheets and returns the best matching command. If a query is provided, uses it for searching.

## Alias Information

**Alias for:** `Invoke-NaviBest`

This is an alias that points to the `Invoke-NaviBest` command. Use this alias as a shorthand for the full command name.
