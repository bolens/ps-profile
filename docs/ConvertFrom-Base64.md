# ConvertFrom-Base64

## Synopsis

Decodes base64 input to text.

## Description

Converts base64 encoded string back to readable text.

## Signature

```powershell
ConvertFrom-Base64
```

## Parameters

### -InputObject

The base64 string to decode.
