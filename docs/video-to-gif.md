# video-to-gif

## Synopsis

Converts video to GIF.

## Description

Uses ffmpeg to convert a video file to animated GIF.

## Alias Information

This is an alias that points to the `ConvertFrom-VideoToGif` command. Use this alias as a shorthand for the full command name.
