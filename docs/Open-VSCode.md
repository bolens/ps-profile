# Open-VSCode

## Synopsis

Opens current directory in VS Code.

## Description

Launches VS Code in the current directory if VS Code is available.

## Signature

```powershell
Open-VSCode
```

## Parameters

No parameters.

No examples provided.

- `vsc` - Opens current directory in VS Code.
