# Test-HasCommand

## Synopsis

Tests if a command is available.

## Description

Utility function to check if a command exists. Uses fast provider checks first
    to avoid module autoload, then falls back to cached or direct command testing.

## Signature

```powershell
Test-HasCommand
```

No parameters.
