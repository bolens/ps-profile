# Test-ProfileFragmentEnabled

## Synopsis

Tests if a profile fragment is enabled.

## Description

Checks the fragment configuration to determine if a fragment is enabled. Fragments are enabled by default unless explicitly disabled.

## Signature

```powershell
Test-ProfileFragmentEnabled
```

## Parameters

### -FragmentName

The name of the fragment to check (e.g., '11-git.ps1' or '11-git').
