# Initialize-Starship

## Synopsis

Initializes the Starship prompt for PowerShell.

## Description

Sets up Starship as the PowerShell prompt if the starship command is available. Uses the standard starship initialization which automatically reads starship.toml.

## Signature

```powershell
Initialize-Starship
```

No parameters.
