# Show-RecentCommands

## Synopsis

Shows recent commands with quick selection.

## Description

Displays recent commands with numbers for quick execution.

## Signature

```powershell
Show-RecentCommands
```

## Parameters

### -Count

Number of recent commands to show (default: 10).
