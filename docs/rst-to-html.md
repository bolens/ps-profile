# rst-to-html

## Synopsis

Converts RST file to HTML.

## Description

Uses pandoc to convert a reStructuredText (RST) file to HTML format.

## Alias Information

**Alias for:** `ConvertTo-HtmlFromRst`

This is an alias that points to the `ConvertTo-HtmlFromRst` command. Use this alias as a shorthand for the full command name.
