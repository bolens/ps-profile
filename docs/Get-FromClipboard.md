# Get-FromClipboard

## Synopsis

Pastes content from the clipboard.

## Description

Retrieves content from the clipboard. Uses Get-Clipboard if available, otherwise falls back to the 'paste' command.

## Signature

```powershell
Get-FromClipboard
```

## Parameters

No parameters.
