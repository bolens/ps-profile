# navis

## Synopsis

Searches navi cheatsheets interactively.

## Description

Launches navi in interactive search mode. If a query is provided, pre-fills the search.

## Alias Information

**Alias for:** `Invoke-NaviSearch`

This is an alias that points to the `Invoke-NaviSearch` command. Use this alias as a shorthand for the full command name.
