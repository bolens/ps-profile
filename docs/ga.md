# ga

## Synopsis

Stages changes for commit.

## Description

Adds file changes to the staging area for the next commit. Forwards all arguments to git add.

## Alias Information

**Alias for:** `Add-GitChanges`

This is an alias that points to the `Add-GitChanges` command. Use this alias as a shorthand for the full command name.
