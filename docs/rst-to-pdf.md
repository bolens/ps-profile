# rst-to-pdf

## Synopsis

Converts RST file to PDF.

## Description

Uses pandoc to convert a reStructuredText (RST) file to PDF format.

## Alias Information

**Alias for:** `ConvertTo-PdfFromRst`

This is an alias that points to the `ConvertTo-PdfFromRst` command. Use this alias as a shorthand for the full command name.
