# Test-IsWindows

## Synopsis

Checks if the current platform is Windows.

## Description

Returns true if running on Windows, false otherwise.

## Signature

```powershell
Test-IsWindows
```

## Parameters

No parameters.

## Outputs

System.Boolean. True if running on Windows, false otherwise.
