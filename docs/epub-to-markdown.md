# epub-to-markdown

## Synopsis

Converts EPUB file to Markdown.

## Description

Uses pandoc to convert an EPUB file to Markdown format.

## Signature

```powershell
epub-to-markdown
```

## Parameters

### -InputPath

The path to the EPUB file.
