# ConvertTo-CsvFromJson

## Synopsis

Converts JSON file to CSV format.

## Description

Parses a JSON file containing an array of objects and converts it to CSV.

## Signature

```powershell
ConvertTo-CsvFromJson
```

## Parameters

### -Path

The path to the JSON file to convert.
