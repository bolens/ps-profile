# Invoke-NaviPrint

## Synopsis

Prints commands from navi cheatsheets without executing them.

## Description

Searches navi cheatsheets and prints the selected command without executing it. If a query is provided, uses it for searching.

## Signature

```powershell
Invoke-NaviPrint
```

## Parameters

No parameters.
