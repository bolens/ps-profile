# mix-install

## Synopsis

Installs Mix dependencies.

## Description

Installs dependencies defined in mix.exs. This is equivalent to running 'mix deps.get'.

## Alias Information

**Alias for:** `Install-MixDependencies`

Use this alias as a shorthand for the full command name.
