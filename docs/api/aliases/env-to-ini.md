# env-to-ini

## Synopsis

Converts a .env file to INI format.

## Description

Converts a .env file to INI format. All key-value pairs are placed in an [env] section.

## Alias Information

**Alias for:** `ConvertFrom-EnvToIni`

Use this alias as a shorthand for the full command name.
