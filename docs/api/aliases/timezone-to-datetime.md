# timezone-to-datetime

## Synopsis

Converts a DateTime from a specific timezone to local time.

## Description

Converts a DateTime object (assumed to be in the specified timezone) to local time.

## Alias Information

**Alias for:** `ConvertFrom-TimeZone`

Use this alias as a shorthand for the full command name.
