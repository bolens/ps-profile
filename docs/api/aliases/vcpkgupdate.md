# vcpkgupdate

## Synopsis

Upgrades vcpkg packages.

## Description

Upgrades specified packages or all packages if no arguments provided.

## Alias Information

**Alias for:** `Update-VcpkgPackages`

Use this alias as a shorthand for the full command name.
