# json-to-bson

## Synopsis

Converts JSON file to BSON format.

## Description

Converts a JSON file to BSON (Binary JSON) format. Requires Node.js and the bson package to be installed.

## Alias Information

**Alias for:** `ConvertTo-BsonFromJson`

Use this alias as a shorthand for the full command name.
