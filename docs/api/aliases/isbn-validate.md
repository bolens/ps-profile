# isbn-validate

## Synopsis

Tests whether an ISBN is valid.

## Description

Returns $true when the input can be parsed and passes ISBN-10 or ISBN-13 checksum validation.

## Alias Information

**Alias for:** `Test-IsbnValid`

Use this alias as a shorthand for the full command name.
