# png-to-ico

## Synopsis

Converts PNG image to ICO format.

## Description

Converts a PNG image file to ICO format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertTo-IcoFromPng`

Use this alias as a shorthand for the full command name.
