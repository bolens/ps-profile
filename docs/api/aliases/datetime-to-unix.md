# datetime-to-unix

## Synopsis

Converts a DateTime object to a Unix timestamp.

## Description

Converts a DateTime object to a Unix timestamp (seconds since January 1, 1970 UTC).

## Alias Information

**Alias for:** `ConvertTo-UnixTimestampFromDateTime`

Use this alias as a shorthand for the full command name.
