# notion-to-markdown

## Synopsis

Exports a Notion page to markdown using notion2md or notionify-cli.

## Description

Wraps available Notion export CLIs. Requires NOTION_TOKEN or -Token.

## Alias Information

**Alias for:** `Export-NotionPageToMarkdown`

Use this alias as a shorthand for the full command name.
