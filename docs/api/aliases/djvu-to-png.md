# djvu-to-png

## Synopsis

Converts DjVu file to PNG.

## Description

Converts a DjVu document file to PNG image format using ImageMagick or djvulibre tools.

## Alias Information

**Alias for:** `ConvertFrom-DjvuToPng`

Use this alias as a shorthand for the full command name.
