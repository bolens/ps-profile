# qrcode-email

## Synopsis

Generates an email QR code.

## Description

Creates a QR code that can be scanned to compose an email. Requires Node.js and qrcode package.

## Alias Information

**Alias for:** `New-QrCodeEmail`

Use this alias as a shorthand for the full command name.
