# notion-callouts-to-obsidian

## Synopsis

Converts Notion-style callout blockquotes to Obsidian callout syntax.

## Description

Rewrites blockquotes such as "> **Note**" or "> 💡 Tip" into Obsidian callouts like "> [!NOTE]" and "> [!TIP]".

## Alias Information

**Alias for:** `Convert-NotionCalloutsToObsidian`

Use this alias as a shorthand for the full command name.
