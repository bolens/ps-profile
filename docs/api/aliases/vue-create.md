# vue-create

## Synopsis

Creates a new Vue.js project.

## Description

Wrapper for Vue CLI create command. Prefers npx @vue/cli, falls back to globally installed vue.

## Alias Information

**Alias for:** `New-VueApp`

Use this alias as a shorthand for the full command name.
