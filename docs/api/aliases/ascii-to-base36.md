# ascii-to-base36

## Synopsis

Converts ASCII text to Base36 encoding.

## Description

Encodes ASCII/UTF-8 text to Base36 format. Base36 is an alphanumeric encoding using 0-9 and A-Z.

## Alias Information

**Alias for:** `ConvertFrom-AsciiToBase36`

Use this alias as a shorthand for the full command name.
