# fb2-to-docx

## Synopsis

Converts FB2 file to DOCX.

## Description

Uses pandoc to convert a FictionBook (FB2) e-book file to Microsoft Word DOCX format.

## Alias Information

**Alias for:** `ConvertTo-DocxFromFb2`

Use this alias as a shorthand for the full command name.
