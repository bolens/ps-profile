# uvtoolupgrade

## Synopsis

Upgrades all globally installed uv tools to their latest versions.

## Description

Upgrades all Python tools that were installed globally using uv tool install. This is equivalent to running 'uv tool upgrade --all'.

## Alias Information

**Alias for:** `Update-UVTools`

Use this alias as a shorthand for the full command name.
