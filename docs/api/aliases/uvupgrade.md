# uvupgrade

## Synopsis

Upgrades all outdated Python packages using uv.

## Description

Lists all outdated packages and upgrades them to their latest versions. This is a uv-compatible replacement for pip-based upgrade commands.

## Alias Information

**Alias for:** `Update-UVOutdatedPackages`

Use this alias as a shorthand for the full command name.
