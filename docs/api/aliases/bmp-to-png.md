# bmp-to-png

## Synopsis

Converts BMP image to PNG format.

## Description

Converts a BMP image file to PNG format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-BmpToPng`

Use this alias as a shorthand for the full command name.
