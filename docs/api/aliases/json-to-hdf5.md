# json-to-hdf5

## Synopsis

Converts JSON file to HDF5 format.

## Description

Converts a JSON file to HDF5 (Hierarchical Data Format version 5) format. Requires Python with h5py package to be installed.

## Alias Information

**Alias for:** `ConvertTo-Hdf5FromJson`

Use this alias as a shorthand for the full command name.
