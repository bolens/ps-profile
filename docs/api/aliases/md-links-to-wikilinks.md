# md-links-to-wikilinks

## Synopsis

Converts markdown link syntax to Obsidian wikilinks.

## Description

Transforms [text](page.md) and [text](page.md#anchor) into [[page|text]] or [[page#anchor|text]]. Skips absolute URLs and non-markdown targets.

## Alias Information

**Alias for:** `ConvertTo-WikilinksFromMarkdownLinks`

Use this alias as a shorthand for the full command name.
