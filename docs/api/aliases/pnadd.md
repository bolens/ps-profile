# pnadd

## Synopsis

Installs packages using pnpm.

## Description

Adds packages as dependencies to the project using pnpm. Supports -D (dev) and -g (global) flags.

## Alias Information

**Alias for:** `Invoke-PnpmInstall`

Use this alias as a shorthand for the full command name.
