# zlib-decompress

## Synopsis

Decompresses a Zlib-compressed file.

## Description

Decompresses a file that was compressed using Zlib (Deflate) compression algorithm.

## Alias Information

**Alias for:** `Expand-Zlib`

Use this alias as a shorthand for the full command name.
