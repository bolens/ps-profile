# rst-to-docx

## Synopsis

Converts RST file to DOCX.

## Description

Uses pandoc to convert a reStructuredText (RST) file to Microsoft Word DOCX format.

## Alias Information

**Alias for:** `ConvertTo-DocxFromRst`

Use this alias as a shorthand for the full command name.
