# regex-catalog-show

## Synopsis

Displays natural language regex catalog entries in a table.

## Description

Shows catalog entries with names, aliases, and notes. Optionally filters by query.

## Alias Information

**Alias for:** `Show-RegexDescriptionCatalog`

Use this alias as a shorthand for the full command name.
