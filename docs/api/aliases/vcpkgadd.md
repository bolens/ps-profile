# vcpkgadd

## Synopsis

Installs C++ libraries using vcpkg.

## Description

Installs packages from vcpkg registry. Supports --triplet and --version flags.

## Alias Information

**Alias for:** `Install-VcpkgPackage`

Use this alias as a shorthand for the full command name.
