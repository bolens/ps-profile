# db-restore

## Synopsis

Restores a database from a backup.

## Description

Restores a database from a backup file using database-specific restore tools. Supports PostgreSQL, MySQL, SQLite, and MongoDB.

## Alias Information

**Alias for:** `Restore-Database`

Use this alias as a shorthand for the full command name.
