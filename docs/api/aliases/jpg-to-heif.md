# jpg-to-heif

## Synopsis

Converts JPEG image to HEIC format.

## Description

Converts a JPEG image file to HEIC format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertTo-HeicFromJpeg`

Use this alias as a shorthand for the full command name.
