# braille-to-ascii

## Synopsis

Converts Braille encoding to ASCII text.

## Description

Decodes Unicode Braille patterns back to ASCII text. Supports standard 6-dot Braille patterns.

## Alias Information

**Alias for:** `ConvertFrom-BrailleToAscii`

Use this alias as a shorthand for the full command name.
