# toon-to-toml

## Synopsis

Converts TOON file to TOML format.

## Description

Converts a TOON (Token-Oriented Object Notation) file to TOML (Tom's Obvious, Minimal Language) format.

## Alias Information

Use this alias as a shorthand for the full command name.
