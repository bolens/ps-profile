# docx-to-odt

## Synopsis

Converts DOCX file to ODT.

## Description

Uses pandoc to convert a Microsoft Word DOCX file to ODT (OpenDocument Text) format.

## Alias Information

**Alias for:** `ConvertTo-OdtFromDocx`

Use this alias as a shorthand for the full command name.
