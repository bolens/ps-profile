# json-to-jsonl

## Synopsis

Converts JSON file to JSONL format.

## Description

Converts a JSON file (array or object) to JSONL (JSON Lines) format, with one JSON object per line.

## Alias Information

**Alias for:** `ConvertTo-JsonLFromJson`

Use this alias as a shorthand for the full command name.
