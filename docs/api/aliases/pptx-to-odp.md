# pptx-to-odp

## Synopsis

Converts PPTX file to ODP.

## Description

Uses pandoc to convert a Microsoft PowerPoint PPTX file to ODP (OpenDocument Presentation) format.

## Alias Information

**Alias for:** `ConvertTo-OdpFromPptx`

Use this alias as a shorthand for the full command name.
