# ascii-to-ebcdic

## Synopsis

Converts ASCII text to EBCDIC encoding.

## Description

Encodes ASCII text to EBCDIC format (Code Page 037). Returns the EBCDIC encoding as a hexadecimal string.

## Alias Information

**Alias for:** `ConvertFrom-AsciiToEBCDIC`

Use this alias as a shorthand for the full command name.
