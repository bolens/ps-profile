# svg-to-pdf

## Synopsis

Converts SVG image to PDF format.

## Description

Converts an SVG image file to PDF format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-SvgToPdf`

Use this alias as a shorthand for the full command name.
