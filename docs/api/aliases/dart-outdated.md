# dart-outdated

## Synopsis

Checks for outdated Dart packages.

## Description

Lists all packages that have newer versions available. This is equivalent to running 'dart pub outdated'.

## Alias Information

**Alias for:** `Test-DartOutdated`

Use this alias as a shorthand for the full command name.
