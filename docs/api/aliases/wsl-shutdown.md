# wsl-shutdown

## Synopsis

Shuts down all WSL distributions.

## Description

Terminates all running WSL distributions and shuts down the WSL subsystem.

## Alias Information

**Alias for:** `Stop-WSL`

Use this alias as a shorthand for the full command name.
