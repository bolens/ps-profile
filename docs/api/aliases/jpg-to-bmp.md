# jpg-to-bmp

## Synopsis

Converts JPEG image to BMP format.

## Description

Converts a JPEG image file to BMP format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertTo-BmpFromJpeg`

Use this alias as a shorthand for the full command name.
