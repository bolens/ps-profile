# tiff-to-png

## Synopsis

Converts TIFF image to PNG format.

## Description

Converts a TIFF image file to PNG format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-TiffToPng`

Use this alias as a shorthand for the full command name.
