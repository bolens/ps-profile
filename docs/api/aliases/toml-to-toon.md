# toml-to-toon

## Synopsis

Converts TOML file to TOON format.

## Description

Converts a TOML (Tom's Obvious, Minimal Language) file to TOON (Token-Oriented Object Notation) format.

## Alias Information

Use this alias as a shorthand for the full command name.
