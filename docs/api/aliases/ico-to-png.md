# ico-to-png

## Synopsis

Converts ICO image to PNG format.

## Description

Converts an ICO image file to PNG format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-IcoToPng`

Use this alias as a shorthand for the full command name.
