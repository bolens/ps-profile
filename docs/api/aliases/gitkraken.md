# gitkraken

## Synopsis

Launches GitKraken GUI.

## Description

Opens GitKraken, a cross-platform Git GUI client, in the current directory or specified repository path.

## Alias Information

**Alias for:** `Invoke-GitKraken`

Use this alias as a shorthand for the full command name.
