# djvu-to-pdf

## Synopsis

Converts DjVu file to PDF.

## Description

Converts a DjVu document file to PDF format using ImageMagick or djvulibre tools.

## Alias Information

**Alias for:** `ConvertFrom-DjvuToPdf`

Use this alias as a shorthand for the full command name.
