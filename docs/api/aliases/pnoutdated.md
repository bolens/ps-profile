# pnoutdated

## Synopsis

Checks for outdated packages in the current project.

## Description

Lists all packages that have newer versions available. This is equivalent to running 'pnpm outdated'.

## Alias Information

**Alias for:** `Test-PnpmOutdated`

Use this alias as a shorthand for the full command name.
