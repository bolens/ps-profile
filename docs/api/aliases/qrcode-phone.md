# qrcode-phone

## Synopsis

Generates a phone call QR code.

## Description

Creates a QR code that can be scanned to make a phone call. Requires Node.js and qrcode package.

## Alias Information

**Alias for:** `New-QrCodePhone`

Use this alias as a shorthand for the full command name.
