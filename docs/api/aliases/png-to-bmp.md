# png-to-bmp

## Synopsis

Converts PNG image to BMP format.

## Description

Converts a PNG image file to BMP format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertTo-BmpFromPng`

Use this alias as a shorthand for the full command name.
