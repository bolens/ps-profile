# jest

## Synopsis

Executes Jest test runner.

## Description

Wrapper for jest command. Uses globally installed jest if available, otherwise falls back to npx.

## Alias Information

**Alias for:** `Invoke-Jest`

Use this alias as a shorthand for the full command name.
