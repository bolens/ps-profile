# base32-to-ascii

## Synopsis

Converts Base32 string to ASCII text.

## Description

Converts a Base32 string back to ASCII text. Base32 uses the alphabet A-Z, 2-7 (32 characters).

## Alias Information

**Alias for:** `ConvertFrom-Base32ToAscii`

Use this alias as a shorthand for the full command name.
