# rfc3339-to-unix

## Synopsis

Converts an RFC 3339 date/time string to a Unix timestamp.

## Description

Converts an RFC 3339 formatted date/time string to a Unix timestamp (seconds since epoch).

## Alias Information

**Alias for:** `ConvertFrom-Rfc3339ToUnixTimestamp`

Use this alias as a shorthand for the full command name.
