# gradle-outdated

## Synopsis

Checks for outdated Gradle dependencies.

## Description

Lists all dependencies that have newer versions available. This requires the gradle-versions-plugin to be configured in build.gradle.

## Alias Information

**Alias for:** `Test-GradleOutdated`

Use this alias as a shorthand for the full command name.
