# pdf-to-tif

## Synopsis

Converts PDF to TIFF format.

## Description

Converts a PDF file to TIFF format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertTo-TiffFromPdf`

Use this alias as a shorthand for the full command name.
