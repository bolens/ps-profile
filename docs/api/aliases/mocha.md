# mocha

## Synopsis

Executes Mocha test runner.

## Description

Wrapper for mocha command. Uses globally installed mocha if available, otherwise falls back to npx.

## Alias Information

**Alias for:** `Invoke-Mocha`

Use this alias as a shorthand for the full command name.
