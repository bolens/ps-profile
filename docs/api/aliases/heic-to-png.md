# heic-to-png

## Synopsis

Converts HEIC/HEIF image to PNG format.

## Description

Converts a HEIC/HEIF image file to PNG format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-HeicToPng`

Use this alias as a shorthand for the full command name.
