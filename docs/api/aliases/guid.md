# guid

## Synopsis

Generates a new UUID (GUID).

## Description

Generates a new UUID (Universally Unique Identifier) using .NET Guid.NewGuid(). Can return the UUID in various formats.

## Alias Information

**Alias for:** `New-Uuid`

Use this alias as a shorthand for the full command name.
