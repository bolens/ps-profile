# ico-to-jpeg

## Synopsis

Converts ICO image to JPEG format.

## Description

Converts an ICO image file to JPEG format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-IcoToJpeg`

Use this alias as a shorthand for the full command name.
