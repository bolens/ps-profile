# isbn-cover

## Synopsis

Downloads a book cover image for an ISBN.

## Description

Looks up the ISBN, resolves the cover URL, and saves the image to disk.

## Alias Information

**Alias for:** `Save-IsbnCover`

Use this alias as a shorthand for the full command name.
