# http-server

## Synopsis

Starts HTTP server.

## Description

Wrapper for http-server command. Uses globally installed http-server if available, otherwise falls back to npx.

## Alias Information

**Alias for:** `Invoke-HttpServer`

Use this alias as a shorthand for the full command name.
