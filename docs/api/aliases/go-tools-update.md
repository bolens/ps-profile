# go-tools-update

## Synopsis

Updates Go development tools to their latest versions.

## Description

Updates all Go tools from golang.org/x/tools to their latest versions. This is equivalent to running 'go install golang.org/x/tools/cmd/...@latest'.

## Alias Information

**Alias for:** `Update-GoTools`

Use this alias as a shorthand for the full command name.
