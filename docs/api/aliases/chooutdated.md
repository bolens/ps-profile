# chooutdated

## Synopsis

Checks for outdated Chocolatey packages.

## Description

Lists all installed packages that have newer versions available. This is equivalent to running 'choco outdated'.

## Alias Information

**Alias for:** `Test-ChocoOutdated`

Use this alias as a shorthand for the full command name.
