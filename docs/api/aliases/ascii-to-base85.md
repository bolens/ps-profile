# ascii-to-base85

## Synopsis

Converts ASCII text to Base85/Ascii85 encoding.

## Description

Encodes ASCII/UTF-8 text to Base85 format. Base85 is commonly used in PDF and PostScript files.

## Alias Information

**Alias for:** `ConvertFrom-AsciiToBase85`

Use this alias as a shorthand for the full command name.
