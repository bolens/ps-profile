# fcmd

## Synopsis

Finds PowerShell commands using fzf fuzzy finder.

## Description

Lists all available commands and uses fzf to interactively select one.

## Alias Information

**Alias for:** `Find-CommandFuzzy`

Use this alias as a shorthand for the full command name.
