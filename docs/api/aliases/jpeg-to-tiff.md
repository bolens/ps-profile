# jpeg-to-tiff

## Synopsis

Converts JPEG image to TIFF format.

## Description

Converts a JPEG image file to TIFF format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertTo-TiffFromJpeg`

Use this alias as a shorthand for the full command name.
