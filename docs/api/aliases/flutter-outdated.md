# flutter-outdated

## Synopsis

Checks for outdated Flutter packages.

## Description

Lists all packages that have newer versions available. This is equivalent to running 'flutter pub outdated'.

## Alias Information

**Alias for:** `Test-FlutterOutdated`

Use this alias as a shorthand for the full command name.
