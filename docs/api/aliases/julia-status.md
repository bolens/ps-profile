# julia-status

## Synopsis

Shows Julia package status.

## Description

Lists all installed packages and their versions. This is equivalent to running 'julia -e "using Pkg; Pkg.status()"'.

## Alias Information

**Alias for:** `Get-JuliaPackages`

Use this alias as a shorthand for the full command name.
