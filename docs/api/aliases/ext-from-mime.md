# ext-from-mime

## Synopsis

Gets file extension(s) from a MIME type.

## Description

Returns the file extension(s) associated with a given MIME type.

## Alias Information

**Alias for:** `Get-ExtensionFromMimeType`

Use this alias as a shorthand for the full command name.
