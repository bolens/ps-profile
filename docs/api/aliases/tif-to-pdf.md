# tif-to-pdf

## Synopsis

Converts TIFF image to PDF format.

## Description

Converts a TIFF image file to PDF format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-TiffToPdf`

Use this alias as a shorthand for the full command name.
