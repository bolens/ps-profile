# jpg-to-ico

## Synopsis

Converts JPEG image to ICO format.

## Description

Converts a JPEG image file to ICO format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertTo-IcoFromJpeg`

Use this alias as a shorthand for the full command name.
