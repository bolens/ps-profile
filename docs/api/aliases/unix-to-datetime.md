# unix-to-datetime

## Synopsis

Converts a Unix timestamp to a DateTime object.

## Description

Converts a Unix timestamp (seconds since January 1, 1970 UTC) to a DateTime object.

## Alias Information

**Alias for:** `ConvertFrom-UnixTimestampToDateTime`

Use this alias as a shorthand for the full command name.
