# djvu-to-jpg

## Synopsis

Converts DjVu file to JPEG.

## Description

Converts a DjVu document file to JPEG image format using ImageMagick or djvulibre tools.

## Alias Information

**Alias for:** `ConvertFrom-DjvuToJpeg`

Use this alias as a shorthand for the full command name.
