# tif-to-jpg

## Synopsis

Converts TIFF image to JPEG format.

## Description

Converts a TIFF image file to JPEG format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-TiffToJpeg`

Use this alias as a shorthand for the full command name.
