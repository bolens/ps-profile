# turbo

## Synopsis

Executes Turbo commands.

## Description

Wrapper for turbo command. Uses globally installed turbo if available, otherwise falls back to npx.

## Alias Information

**Alias for:** `Invoke-Turbo`

Use this alias as a shorthand for the full command name.
