# winget-cleanup

## Synopsis

Cleans up winget cache.

## Description

Removes cached installer packages from winget's cache directory. This helps free up disk space by removing downloaded installers.

## Alias Information

**Alias for:** `Clear-WingetCache`

Use this alias as a shorthand for the full command name.
