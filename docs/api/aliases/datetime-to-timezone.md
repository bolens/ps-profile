# datetime-to-timezone

## Synopsis

Converts a DateTime to a specific timezone.

## Description

Converts a DateTime object (assumed to be in local time) to a specific timezone.

## Alias Information

**Alias for:** `ConvertTo-TimeZone`

Use this alias as a shorthand for the full command name.
