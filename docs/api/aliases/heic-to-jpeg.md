# heic-to-jpeg

## Synopsis

Converts HEIC/HEIF image to JPEG format.

## Description

Converts a HEIC/HEIF image file to JPEG format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-HeicToJpeg`

Use this alias as a shorthand for the full command name.
