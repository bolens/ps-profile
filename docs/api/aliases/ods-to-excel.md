# ods-to-excel

## Synopsis

Converts ODS file to Excel.

## Description

Uses pandoc to convert an ODS (OpenDocument Spreadsheet) file to Excel (XLSX) format.

## Alias Information

**Alias for:** `ConvertTo-ExcelFromOds`

Use this alias as a shorthand for the full command name.
