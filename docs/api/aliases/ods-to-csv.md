# ods-to-csv

## Synopsis

Converts ODS file to CSV.

## Description

Uses pandoc or LibreOffice to convert an ODS (OpenDocument Spreadsheet) file to CSV format.

## Alias Information

**Alias for:** `ConvertFrom-OdsToCsv`

Use this alias as a shorthand for the full command name.
