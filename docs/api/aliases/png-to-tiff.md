# png-to-tiff

## Synopsis

Converts PNG image to TIFF format.

## Description

Converts a PNG image file to TIFF format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertTo-TiffFromPng`

Use this alias as a shorthand for the full command name.
