# ebcdic-to-ascii

## Synopsis

Converts EBCDIC encoding to ASCII text.

## Description

Decodes EBCDIC encoded string (as hexadecimal) back to ASCII text.

## Alias Information

**Alias for:** `ConvertFrom-EBCDICToAscii`

Use this alias as a shorthand for the full command name.
