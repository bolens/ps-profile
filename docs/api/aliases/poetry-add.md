# poetry-add

## Synopsis

Adds dependencies to Poetry project.

## Description

Adds packages as dependencies to pyproject.toml. Supports --group dev, --group test, --group docs flags.

## Alias Information

**Alias for:** `Add-PoetryDependency`

Use this alias as a shorthand for the full command name.
