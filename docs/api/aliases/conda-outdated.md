# conda-outdated

## Synopsis

Checks for outdated Conda packages.

## Description

Lists all packages that have newer versions available. This is equivalent to running 'conda list --outdated'.

## Alias Information

**Alias for:** `Test-CondaOutdated`

Use this alias as a shorthand for the full command name.
