# xlsx-to-csv

## Synopsis

Converts Excel file to CSV.

## Description

Uses ImportExcel module or pandoc to convert an Excel (XLSX/XLS) file to CSV format.

## Alias Information

**Alias for:** `ConvertFrom-ExcelToCsv`

Use this alias as a shorthand for the full command name.
