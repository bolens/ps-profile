# png-to-heif

## Synopsis

Converts PNG image to HEIC format.

## Description

Converts a PNG image file to HEIC format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertTo-HeicFromPng`

Use this alias as a shorthand for the full command name.
