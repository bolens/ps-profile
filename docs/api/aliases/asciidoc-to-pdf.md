# asciidoc-to-pdf

## Synopsis

Converts AsciiDoc file to PDF.

## Description

Uses pandoc or asciidoctor-pdf to convert an AsciiDoc file to PDF format.

## Alias Information

**Alias for:** `ConvertFrom-AsciidocToPdf`

Use this alias as a shorthand for the full command name.
