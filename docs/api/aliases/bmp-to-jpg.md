# bmp-to-jpg

## Synopsis

Converts BMP image to JPEG format.

## Description

Converts a BMP image file to JPEG format using ImageMagick or GraphicsMagick.

## Alias Information

**Alias for:** `ConvertFrom-BmpToJpeg`

Use this alias as a shorthand for the full command name.
