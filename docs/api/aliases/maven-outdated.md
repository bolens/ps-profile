# maven-outdated

## Synopsis

Checks for outdated Maven dependencies.

## Description

Lists all dependencies that have newer versions available. This uses the versions-maven-plugin to display dependency updates.

## Alias Information

**Alias for:** `Test-MavenOutdated`

Use this alias as a shorthand for the full command name.
