# julia-add

## Synopsis

Adds Julia packages.

## Description

Adds packages to the current Julia environment. This is equivalent to running 'julia -e "using Pkg; Pkg.add([\"package\"])"'.

## Alias Information

**Alias for:** `Add-JuliaPackage`

Use this alias as a shorthand for the full command name.
