# isbn-editions

## Synopsis

Lists alternate editions for a book ISBN.

## Description

Resolves the Open Library work for an ISBN and returns edition metadata rows.

## Alias Information

**Alias for:** `Get-IsbnEditions`

Use this alias as a shorthand for the full command name.
