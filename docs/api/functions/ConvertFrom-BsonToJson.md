# ConvertFrom-BsonToJson

## Synopsis

Converts BSON file to JSON format.

## Description

Converts a BSON (Binary JSON) file back to JSON format. Requires Node.js and the bson package to be installed.

## Signature

```powershell
ConvertFrom-BsonToJson
```

## Parameters

### -InputPath

The path to the BSON file.

### -OutputPath

The path for the output JSON file. If not specified, uses input path with .json extension.
