# Test-BrewOutdated

## Synopsis

Checks for outdated Homebrew packages.

## Description

Lists all installed packages that have newer versions available. This is equivalent to running 'brew outdated'.

## Signature

```powershell
Test-BrewOutdated
```

## Parameters

No parameters.
