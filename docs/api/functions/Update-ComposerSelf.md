# Update-ComposerSelf

## Synopsis

Updates Composer to the latest version.

## Description

Updates Composer itself to the latest version using 'composer self-update'.

## Signature

```powershell
Update-ComposerSelf
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Update-ComposerSelf
    Updates Composer to the latest version.
``
