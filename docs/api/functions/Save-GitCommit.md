# Save-GitCommit

## Synopsis

Commits staged changes.

## Description

Creates a new commit with the currently staged changes. Forwards all arguments to git commit.

## Signature

```powershell
Save-GitCommit [Object]$a
```

## Parameters

No parameters.

## Examples

No examples provided.

- `gc` - Commits staged changes.
