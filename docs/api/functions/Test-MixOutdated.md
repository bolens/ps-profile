# Test-MixOutdated

## Synopsis

Checks for outdated Elixir dependencies.

## Description

Lists all dependencies that have newer versions available. This is equivalent to running 'mix deps.outdated'.

## Signature

```powershell
Test-MixOutdated
```

No parameters.
