# Add-UVDependency

## Synopsis

Adds dependencies to UV project.

## Description

Adds packages as dependencies to the current UV project.

## Signature

```powershell
Add-UVDependency
```

## Parameters

### -Arguments

Arguments forwarded to uv pip.


## Examples

### Example 1

`powershell
Add-UVDependency
.PARAMETER Arguments
    Arguments forwarded to uv pip.
``

## Aliases

This function has the following aliases:

- `uva` - Adds dependencies to UV project.
