# Invoke-MojoRun

## Synopsis

Runs Mojo programs.

## Description

Executes Mojo source files.

## Signature

```powershell
Invoke-MojoRun
```

## Parameters

### -Files

Mojo source files to execute.


## Examples

### Example 1

`powershell
Invoke-MojoRun ./main.mojo
.PARAMETER Files
    Mojo source files to execute.
``

## Aliases

This function has the following aliases:

- `mojo-run` - Runs Mojo programs.
