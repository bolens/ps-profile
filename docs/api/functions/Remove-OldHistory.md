# Remove-OldHistory

## Synopsis

Removes old commands from history.

## Description

Removes commands older than the specified number of days from PowerShell history.

## Signature

```powershell
Remove-OldHistory
```

## Parameters

### -Days

Number of days of history to keep (default: 30).


## Examples

### Example 1

`powershell
Remove-OldHistory
``
