# ConvertFrom-SuperJsonToToml

## Synopsis

Converts SuperJSON file to TOML format.

## Description

Converts a SuperJSON file to TOML (Tom's Obvious, Minimal Language) format. Requires Node.js, superjson package, yq command, and PSToml module.

## Signature

```powershell
ConvertFrom-SuperJsonToToml
```

## Parameters

### -InputPath

The path to the SuperJSON file.

### -OutputPath

The path for the output TOML file. If not specified, uses input path with .toml extension.
