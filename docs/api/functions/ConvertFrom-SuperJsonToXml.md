# ConvertFrom-SuperJsonToXml

## Synopsis

Converts SuperJSON file to XML format.

## Description

Converts a SuperJSON file to XML format. Requires Node.js and superjson package.

## Signature

```powershell
ConvertFrom-SuperJsonToXml
```

## Parameters

### -InputPath

The path to the SuperJSON file.

### -OutputPath

The path for the output XML file. If not specified, uses input path with .xml extension.
