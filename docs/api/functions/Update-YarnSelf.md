# Update-YarnSelf

## Synopsis

Updates Yarn to the latest version.

## Description

Updates Yarn itself to the latest version using 'yarn set version latest'.

## Signature

```powershell
Update-YarnSelf
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Update-YarnSelf
    Updates Yarn to the latest version.
``

## Aliases

This function has the following aliases:

- `yarn-update` - Updates Yarn to the latest version.
