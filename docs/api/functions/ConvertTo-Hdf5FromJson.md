# ConvertTo-Hdf5FromJson

## Synopsis

Converts JSON file to HDF5 format.

## Description

Converts a JSON file to HDF5 (Hierarchical Data Format version 5) format. Requires Python with h5py package to be installed.

## Signature

```powershell
ConvertTo-Hdf5FromJson
```

## Parameters

### -InputPath

The path to the JSON file.

### -OutputPath

The path for the output HDF5 file. If not specified, uses input path with .h5 extension.

This function has the following aliases:

- `json-to-h5` - Converts JSON file to HDF5 format.
- `json-to-hdf5` - Converts JSON file to HDF5 format.
