# Compare-GitChanges

## Synopsis

Shows differences between commits, branches, or working tree.

## Description

Displays changes between the working tree and staging area, or between commits. Forwards all arguments to git diff.

## Signature

```powershell
Compare-GitChanges [Object]$a
```

## Parameters

No parameters.
