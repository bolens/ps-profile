# Convert-Audio

## Synopsis

Converts audio file formats.

## Description

Uses ffmpeg to convert an audio file from one format to another.

## Signature

```powershell
Convert-Audio
```

## Parameters

### -InputPath

The path to the input audio file.

### -OutputPath

The path for the output audio file with desired format.
