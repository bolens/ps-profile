# Add-GitChanges

## Synopsis

Stages changes for commit.

## Description

Adds file changes to the staging area for the next commit. Forwards all arguments to git add.

## Signature

```powershell
Add-GitChanges [Object]$a
```

## Parameters

No parameters.

No examples provided.
