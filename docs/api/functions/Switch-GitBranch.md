# Switch-GitBranch

## Synopsis

Switches branches or restores working tree files.

## Description

Changes the active branch or restores files from a specific commit or branch. Forwards all arguments to git checkout.

## Signature

```powershell
Switch-GitBranch
```

## Parameters

No parameters.
