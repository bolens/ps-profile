# Test-DotnetOutdated

## Synopsis

Lists outdated NuGet packages in .NET projects.

## Description

Lists all packages that have newer versions available. This is equivalent to running 'dotnet list package --outdated'.

## Signature

```powershell
Test-DotnetOutdated
```

No parameters.
