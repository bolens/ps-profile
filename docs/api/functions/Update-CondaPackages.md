# Update-CondaPackages

## Synopsis

Updates all packages in the current Conda environment.

## Description

Updates all packages in the current environment to their latest versions. This is equivalent to running 'conda update --all'.

## Signature

```powershell
Update-CondaPackages
```

## Parameters

No parameters.

No examples provided.
