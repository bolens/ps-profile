# Test-PnpmOutdated

## Synopsis

Checks for outdated packages in the current project.

## Description

Lists all packages that have newer versions available. This is equivalent to running 'pnpm outdated'.

## Signature

```powershell
Test-PnpmOutdated
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Test-PnpmOutdated
        Checks for outdated packages in the current project.
``

This function has the following aliases:

- `pnoutdated` - Checks for outdated packages in the current project.
