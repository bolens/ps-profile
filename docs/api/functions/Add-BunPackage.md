# Add-BunPackage

## Synopsis

Adds packages using Bun.

## Description

Wrapper for bun add command.

## Signature

```powershell
Add-BunPackage
```

## Parameters

### -Arguments

Arguments to pass to bun add.


## Examples

### Example 1

`powershell
Add-BunPackage express
``

### Example 2

`powershell
Add-BunPackage -D typescript
``

## Aliases

This function has the following aliases:

- `bun-add` - Updates Bun to the latest version.
