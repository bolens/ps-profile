# Start-NextJsDev

## Synopsis

Starts Next.js development server.

## Description

Wrapper for npx next dev command.

## Signature

```powershell
Start-NextJsDev
```

## Parameters

### -Arguments

Arguments to pass to next dev.


## Examples

### Example 1

`powershell
Start-NextJsDev
``

### Example 2

`powershell
Start-NextJsDev --port 3001
``

## Aliases

This function has the following aliases:

- `next-dev` - Starts Next.js development server.
