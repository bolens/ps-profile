# ConvertFrom-YamlToCsv

## Synopsis

Converts YAML file to CSV format.

## Description

Reads a YAML file and outputs its contents as CSV.

## Signature

```powershell
ConvertFrom-YamlToCsv
```

## Parameters

### -Path

The path to the YAML file to convert.


## Examples

### Example 1

```powershell
ConvertFrom-YamlToCsv -InputPath ./input.file
```

## Aliases

This function has the following aliases:

- `yaml-to-csv` - Converts YAML file to CSV format.
