# New-GitHubPullRequest

## Synopsis

Creates a GitHub pull request.

## Description

Creates a new pull request using the GitHub CLI (gh). Forwards all arguments to gh pr create.

## Signature

```powershell
New-GitHubPullRequest
```

## Parameters

No parameters.

## Examples

No examples provided.

- `prc` - Creates a GitHub pull request.
