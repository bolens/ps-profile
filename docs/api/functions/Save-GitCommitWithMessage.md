# Save-GitCommitWithMessage

## Synopsis

Commits staged changes with a message.

## Description

Creates a new commit with the currently staged changes and the provided commit message. Forwards all arguments to git commit (typically used with -m flag).

## Signature

```powershell
Save-GitCommitWithMessage [Object]$a
```

## Parameters

No parameters.

## Examples

No examples provided.

This function has the following aliases:

- `gcm` - Commits staged changes with a message.
