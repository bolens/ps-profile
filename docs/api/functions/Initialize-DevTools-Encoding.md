# Initialize-DevTools-Encoding

## Synopsis

Initializes URL and HTML encoding utility functions.

## Description

Sets up internal functions for URL and HTML encoding/decoding. This function is called automatically by Ensure-DevTools.

## Signature

```powershell
Initialize-DevTools-Encoding
```

## Parameters

No parameters.

## Examples

No examples provided.

## Notes

This is an internal initialization function and should not be called directly. Requires System.Web assembly for HttpUtility.
