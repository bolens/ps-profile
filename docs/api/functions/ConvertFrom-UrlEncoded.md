# ConvertFrom-UrlEncoded

## Synopsis

URL-decodes a string.

## Description

Decodes a URL-encoded string.

## Signature

```powershell
ConvertFrom-UrlEncoded
```

## Parameters

### -text

URL-encoded text to decode back to plain text.


## Examples

### Example 1

```powershell
ConvertFrom-UrlEncoded -text 'hello%20world'
```
