# Resize-Image

## Synopsis

Resizes an image.

## Description

Uses ImageMagick to resize an image to specified dimensions.

## Signature

```powershell
Resize-Image
```

## Parameters

### -InputPath

The path to the input image file.

### -OutputPath

The path for the output image file.

### -Width

The desired width.

### -Height

The desired height.
