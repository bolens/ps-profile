# Remove-MixDependency

## Synopsis

Removes Mix dependencies.

## Description

Note: Mix dependencies are removed by editing mix.exs. This function provides guidance and cleans dependencies.

## Signature

```powershell
Remove-MixDependency
```

## Parameters

### -Package

Package name to remove.


## Examples

### Example 1

`powershell
Remove-MixDependency phoenix
        Provides instructions for removing Phoenix dependency.
``

## Aliases

This function has the following aliases:

- `mix-remove` - Removes Mix dependencies.
