# Update-JuliaPackages

## Synopsis

Updates Julia packages.

## Description

Updates all packages in the current Julia environment to their latest versions. This is equivalent to running 'julia -e "using Pkg; Pkg.update()"'.

## Signature

```powershell
Update-JuliaPackages
```

## Parameters

No parameters.
