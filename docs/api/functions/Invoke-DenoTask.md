# Invoke-DenoTask

## Synopsis

Runs Deno tasks.

## Description

Wrapper for deno task command to run tasks defined in deno.json.

## Signature

```powershell
Invoke-DenoTask
```

## Parameters

### -Arguments

Arguments to pass to deno task.


## Examples

### Example 1

`powershell
Invoke-DenoTask dev
``

### Example 2

`powershell
Invoke-DenoTask build
``

## Aliases

This function has the following aliases:

- `deno-task` - Runs Deno tasks.
