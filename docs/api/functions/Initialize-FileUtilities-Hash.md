# Initialize-FileUtilities-Hash

## Synopsis

Initializes file hash utility functions.

## Description

Sets up internal functions for file hash operations. This function is called automatically by Ensure-FileUtilities.

## Signature

```powershell
Initialize-FileUtilities-Hash
```

## Parameters

No parameters.

## Examples

No examples provided.

## Notes

This is an internal initialization function and should not be called directly.
