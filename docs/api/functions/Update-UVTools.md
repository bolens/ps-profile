# Update-UVTools

## Synopsis

Upgrades all globally installed uv tools to their latest versions.

## Description

Upgrades all Python tools that were installed globally using uv tool install. This is equivalent to running 'uv tool upgrade --all'.

## Signature

```powershell
Update-UVTools
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Update-UVTools
        Upgrades all globally installed uv tools.
``

## Aliases

This function has the following aliases:

- `uvtoolupgrade` - Upgrades all globally installed uv tools to their latest versions.
