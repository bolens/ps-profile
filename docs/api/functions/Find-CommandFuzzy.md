# Find-CommandFuzzy

## Synopsis

Finds PowerShell commands using fzf fuzzy finder.

## Description

Lists all available commands and uses fzf to interactively select one.

## Signature

```powershell
Find-CommandFuzzy
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Find-CommandFuzzy
``

## Aliases

This function has the following aliases:

- `fcmd` - Finds PowerShell commands using fzf fuzzy finder.
