# ConvertFrom-FitsToJson

## Synopsis

Converts FITS file to JSON format.

## Description

Converts a FITS (Flexible Image Transport System) file to JSON format. FITS is commonly used in astronomy for storing images and data tables. Requires Python with astropy package to be installed.

## Signature

```powershell
ConvertFrom-FitsToJson
```

## Parameters

### -InputPath

The path to the FITS file (.fits or .fit extension).

### -OutputPath

The path for the output JSON file. If not specified, uses input path with .json extension.
