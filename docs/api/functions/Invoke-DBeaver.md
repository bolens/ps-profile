# Invoke-DBeaver

## Synopsis

Launches DBeaver Universal Database Tool.

## Description

Opens DBeaver, a universal database tool that supports many database types.

## Signature

```powershell
Invoke-DBeaver
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Invoke-DBeaver
``

This function has the following aliases:

- `dbeaver` - Launches DBeaver Universal Database Tool.
