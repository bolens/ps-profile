# Update-PnpmSelf

## Synopsis

Updates pnpm to the latest version.

## Description

Updates pnpm itself to the latest version using 'pnpm add -g pnpm@latest'.

## Signature

```powershell
Update-PnpmSelf
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Update-PnpmSelf
        Updates pnpm to the latest version.
``

## Aliases

This function has the following aliases:

- `pnupgrade` - Updates pnpm to the latest version.
