# Build-NextJsApp

## Synopsis

Builds Next.js application for production.

## Description

Wrapper for npx next build command.

## Signature

```powershell
Build-NextJsApp
```

## Parameters

### -Arguments

Arguments to pass to next build.


## Examples

### Example 1

`powershell
Build-NextJsApp
``

### Example 2

`powershell
Build-NextJsApp --no-lint
``

## Aliases

This function has the following aliases:

- `next-build` - Builds Next.js application for production.
