# New-NextJsApp

## Synopsis

Creates a new Next.js application.

## Description

Wrapper for npx create-next-app command.

## Signature

```powershell
New-NextJsApp
```

## Parameters

### -Arguments

Arguments to pass to create-next-app.


## Examples

### Example 1

`powershell
New-NextJsApp my-app
``

### Example 2

`powershell
New-NextJsApp my-app --typescript --tailwind
``

## Aliases

This function has the following aliases:

- `create-next-app` - Creates a new Next.js application.
