# Start-AzureDeveloperUp

## Synopsis

Provisions and deploys using Azure Developer CLI.

## Description

Wrapper for azd up command.

## Signature

```powershell
Start-AzureDeveloperUp
```

## Parameters

### -Arguments

Arguments to pass to azd up.


## Examples

### Example 1

`powershell
Start-AzureDeveloperUp
``

### Example 2

`powershell
Start-AzureDeveloperUp --location eastus
``
