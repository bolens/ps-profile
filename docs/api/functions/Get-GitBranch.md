# Get-GitBranch

## Synopsis

Lists, creates, or deletes branches.

## Description

Manages Git branches. Lists branches when called without arguments, or creates/deletes branches with arguments. Forwards all arguments to git branch.

## Signature

```powershell
Get-GitBranch
```

## Parameters

No parameters.
