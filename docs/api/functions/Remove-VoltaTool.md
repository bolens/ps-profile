# Remove-VoltaTool

## Synopsis

Uninstalls tools from Volta.

## Description

Removes installed tool versions.

## Signature

```powershell
Remove-VoltaTool
```

## Parameters

### -Tools

Tool names with versions to uninstall.


## Examples

### Example 1

`powershell
Remove-VoltaTool node@18
        Uninstalls Node.js 18.
``

## Aliases

This function has the following aliases:

- `voltaremove` - Uninstalls tools from Volta.
- `voltauninstall` - Uninstalls tools from Volta.
