# Open-Explorer

## Synopsis

Opens the current directory in the system file manager.

## Description

Launches the system file manager in the current directory. On Windows uses explorer.exe; on Linux uses xdg-open; on macOS uses open.

## Signature

```powershell
Open-Explorer
```

## Parameters

No parameters.
