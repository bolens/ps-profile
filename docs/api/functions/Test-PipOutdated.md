# Test-PipOutdated

## Synopsis

Checks for outdated Python packages.

## Description

Lists all packages that have newer versions available. This is equivalent to running 'pip list --outdated'.

## Signature

```powershell
Test-PipOutdated
```

## Parameters

No parameters.
