# Find-HistoryQuick

## Synopsis

Quick search in command history.

## Description

Searches command history for the specified pattern.

## Signature

```powershell
Find-HistoryQuick
```

## Parameters

### -Pattern

Text pattern to match against saved command history entries.


## Examples

### Example 1

`powershell
Find-HistoryQuick -Pattern 'git'
.PARAMETER Pattern
    Text pattern to match against saved command history entries.
``

## Aliases

This function has the following aliases:

- `fh` - Quick search in command history.
