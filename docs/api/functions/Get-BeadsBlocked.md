# Get-BeadsBlocked

## Synopsis

Gets blocked issues.

## Description

Returns a list of issues that are blocked by other open issues.

## Signature

```powershell
Get-BeadsBlocked
```

## Parameters

No parameters.

## Outputs

System.String. Output from bd blocked command.


## Examples

### Example 1

`powershell
Get-BeadsBlocked
        Gets all blocked issues.
``
