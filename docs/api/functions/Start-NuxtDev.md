# Start-NuxtDev

## Synopsis

Starts Nuxt.js development server.

## Description

Wrapper for npx nuxi dev command.

## Signature

```powershell
Start-NuxtDev
```

## Parameters

### -Arguments

Arguments to pass to nuxi dev.


## Examples

### Example 1

`powershell
Start-NuxtDev
``

### Example 2

`powershell
Start-NuxtDev --port 3000
``

## Aliases

This function has the following aliases:

- `nuxt-dev` - Starts Nuxt.js development server.
