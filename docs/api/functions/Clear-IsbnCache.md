# Clear-IsbnCache

## Synopsis

Clears cached ISBN lookup results.

## Description

No description available.

## Signature

```powershell
Clear-IsbnCache
```

## Parameters

### -LookupIsbn

Optional ISBN to clear. When omitted, clears the entire ISBN cache directory.

### -Provider

Provider scope for a single ISBN cache entry. Defaults to Auto.
