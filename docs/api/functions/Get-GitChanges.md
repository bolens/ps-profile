# Get-GitChanges

## Synopsis

Fetches and merges changes from remote repository.

## Description

Downloads changes from the remote repository and merges them into the current branch. Forwards all arguments to git pull.

## Signature

```powershell
Get-GitChanges
```

## Parameters

No parameters.
