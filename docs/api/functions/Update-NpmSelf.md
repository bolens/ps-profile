# Update-NpmSelf

## Synopsis

Updates npm to the latest version.

## Description

Updates npm itself to the latest version using 'npm install -g npm@latest'.

## Signature

```powershell
Update-NpmSelf
```

## Parameters

No parameters.

No examples provided.
