# Update-ComposerPackages

## Synopsis

Updates all packages in the current Composer project to their latest versions.

## Description

Updates all packages to their latest versions according to the version constraints specified in composer.json. This is equivalent to running 'composer update'.

## Signature

```powershell
Update-ComposerPackages
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Update-ComposerPackages
    Updates all packages in the current Composer project.
``

## Aliases

This function has the following aliases:

- `composer-update` - Updates all packages in the current Composer project to their latest versions.
