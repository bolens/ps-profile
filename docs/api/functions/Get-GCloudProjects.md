# Get-GCloudProjects

## Synopsis

Manages Google Cloud Platform projects.

## Description

Wrapper for gcloud projects commands.

## Signature

```powershell
Get-GCloudProjects
```

## Parameters

### -Arguments

Arguments to pass to gcloud projects.


## Examples

### Example 1

`powershell
Get-GCloudProjects list
``

### Example 2

`powershell
Get-GCloudProjects describe my-project
``

This function has the following aliases:

- `gcloud-projects` - Manages Google Cloud Platform projects.
