# Build-ViteApp

## Synopsis

Builds Vite application for production.

## Description

Wrapper for npx vite build command.

## Signature

```powershell
Build-ViteApp
```

## Parameters

### -Arguments

Arguments to pass to vite build.


## Examples

### Example 1

`powershell
Build-ViteApp
``

### Example 2

`powershell
Build-ViteApp --mode production
``

## Aliases

This function has the following aliases:

- `vite-build` - Builds Vite application for production.
