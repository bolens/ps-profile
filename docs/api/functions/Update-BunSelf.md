# Update-BunSelf

## Synopsis

Updates Bun to the latest version.

## Description

Updates Bun itself to the latest version using 'bun upgrade'.

## Signature

```powershell
Update-BunSelf
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Update-BunSelf
    Updates Bun to the latest version.
``

## Aliases

This function has the following aliases:

- `bun-upgrade` - Updates Bun to the latest version.
