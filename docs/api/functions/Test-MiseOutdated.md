# Test-MiseOutdated

## Synopsis

Checks for outdated Mise runtimes.

## Description

Lists all installed runtimes that have newer versions available. This is equivalent to running 'mise outdated'.

## Signature

```powershell
Test-MiseOutdated
```

## Parameters

No parameters.

No examples provided.
