# Test-RustupUpdates

## Synopsis

Checks for Rust toolchain updates.

## Description

Checks for available updates to the Rust toolchain without installing them. This is equivalent to running 'rustup check'.

## Signature

```powershell
Test-RustupUpdates
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Test-RustupUpdates
    Checks for available Rust toolchain updates.
``

## Aliases

This function has the following aliases:

- `rustup-check` - Checks for Rust toolchain updates.
