# ConvertFrom-XmlToJson

## Synopsis

Converts XML file to JSON format.

## Description

Parses an XML file and converts it to JSON representation.

## Signature

```powershell
ConvertFrom-XmlToJson
```

## Parameters

### -Path

The path to the XML file to convert.


## Examples

### Example 1

`powershell
ConvertFrom-XmlToJson -InputPath ./input.file
``

## Aliases

This function has the following aliases:

- `xml-to-json` - Converts XML file to JSON format.
