# Convert-Image

## Synopsis

Converts image file formats.

## Description

Uses ImageMagick to convert an image from one format to another.

## Signature

```powershell
Convert-Image
```

## Parameters

### -InputPath

The path to the input image file.

### -OutputPath

The path for the output image file with desired format.
