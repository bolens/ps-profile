# Open-Editor

## Synopsis

Opens file in the best available editor.

## Description

Opens the specified file in the best available editor.

## Signature

```powershell
Open-Editor
```

## Parameters

### -p

The path to the file to open.


## Examples

### Example 1

```powershell
Open-Editor -p 'value'
```

## Aliases

This function has the following aliases:

- `e` - Opens file in the best available editor.
