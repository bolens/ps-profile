# Launch-Blockbench

## Synopsis

Launches Blockbench 3D model editor.

## Description

Launches Blockbench, a 3D model editor for block-based models. Supports Minecraft, Bedrock, and other block-based game formats.

## Signature

```powershell
Launch-Blockbench
```

## Parameters

### -ProjectPath

Optional path to project file to open.


## Outputs

None.


## Examples

### Example 1

`powershell
Launch-Blockbench
        
        Launches Blockbench.
``

### Example 2

`powershell
Launch-Blockbench -ProjectPath "model.bbmodel"
        
        Launches Blockbench and opens a project file.
``
