# Clear-WingetCache

## Synopsis

Cleans up winget cache.

## Description

Removes cached installer packages from winget's cache directory. This helps free up disk space by removing downloaded installers.

## Signature

```powershell
Clear-WingetCache
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Clear-WingetCache
        Cleans the winget cache.
``

## Aliases

This function has the following aliases:

- `winget-clean` - Cleans up winget cache.
- `winget-cleanup` - Cleans up winget cache.
