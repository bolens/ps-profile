# Copy-ToClipboard

## Synopsis

Copies input to the clipboard.

## Description

Copies text or objects to the clipboard. Uses Set-Clipboard if available, otherwise falls back to the 'clip' command.

## Signature

```powershell
Copy-ToClipboard
```

## Parameters

No parameters.
