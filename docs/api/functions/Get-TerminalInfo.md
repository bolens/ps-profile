# Get-TerminalInfo

## Synopsis

Gets information about available terminal emulators.

## Description

Checks for installed terminal emulators and returns information about them. Lists available terminals and their status.

## Signature

```powershell
Get-TerminalInfo
```

## Parameters

No parameters.

## Outputs

System.Object[]. Array of terminal information objects.


## Examples

### Example 1

`powershell
Get-TerminalInfo
        
        Lists all available terminal emulators.
``
