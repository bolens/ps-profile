# New-NuxtApp

## Synopsis

Creates a new Nuxt.js application.

## Description

Wrapper for npx nuxi@latest init command.

## Signature

```powershell
New-NuxtApp
```

## Parameters

### -Arguments

Arguments to pass to nuxi init.


## Examples

### Example 1

`powershell
New-NuxtApp my-app
``

### Example 2

`powershell
New-NuxtApp my-app --package-manager pnpm
``

## Aliases

This function has the following aliases:

- `create-nuxt-app` - Creates a new Nuxt.js application.
