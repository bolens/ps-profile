# Test-ProfileUpdates

## Synopsis

Checks for profile updates and displays changelog.

## Description

Checks if the profile repository has new commits and displays a summary of recent changes. Only shows updates once per day to avoid spam.

## Signature

```powershell
Test-ProfileUpdates
```
