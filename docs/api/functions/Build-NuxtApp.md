# Build-NuxtApp

## Synopsis

Builds Nuxt.js application for production.

## Description

Wrapper for npx nuxi build command.

## Signature

```powershell
Build-NuxtApp
```

## Parameters

### -Arguments

Arguments to pass to nuxi build.


## Examples

### Example 1

`powershell
Build-NuxtApp
``

### Example 2

`powershell
Build-NuxtApp --prerender
``

## Aliases

This function has the following aliases:

- `nuxt-build` - Builds Nuxt.js application for production.
