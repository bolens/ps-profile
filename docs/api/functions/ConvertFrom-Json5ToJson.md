# ConvertFrom-Json5ToJson

## Synopsis

Converts JSON5 file to JSON format.

## Description

Converts a JSON5 file (JSON with comments and trailing commas) to standard JSON format. Requires Node.js and the json5 package to be installed.

## Signature

```powershell
ConvertFrom-Json5ToJson
```

## Parameters

### -InputPath

The path to the JSON5 file.

### -OutputPath

The path for the output JSON file. If not specified, uses input path with .json extension.
