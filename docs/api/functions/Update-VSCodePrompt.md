# Update-VSCodePrompt

## Synopsis

Updates VS Code's prompt state if VS Code is active.

## Description

Updates the VS Code global state with the current prompt function to ensure VS Code's integrated terminal properly tracks the prompt.

## Signature

```powershell
Update-VSCodePrompt
```
