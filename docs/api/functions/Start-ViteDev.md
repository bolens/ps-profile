# Start-ViteDev

## Synopsis

Starts Vite development server.

## Description

Wrapper for npx vite dev command.

## Signature

```powershell
Start-ViteDev
```

## Parameters

### -Arguments

Arguments to pass to vite dev.


## Examples

### Example 1

`powershell
Start-ViteDev
``

### Example 2

`powershell
Start-ViteDev --port 3000
``

## Aliases

This function has the following aliases:

- `vite-dev` - Starts Vite development server.
