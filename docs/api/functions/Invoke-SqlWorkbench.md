# Invoke-SqlWorkbench

## Synopsis

Launches SQL Workbench/J.

## Description

Opens SQL Workbench/J, a universal database tool for SQL databases.

## Signature

```powershell
Invoke-SqlWorkbench
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Invoke-SqlWorkbench
``

This function has the following aliases:

- `sql-workbench` - Launches SQL Workbench/J.
