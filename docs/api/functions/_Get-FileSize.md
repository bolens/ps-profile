# _Get-FileSize

## Synopsis

Initializes file size utility functions.

## Description

Sets up internal functions for file size operations. This function is called automatically by Ensure-FileUtilities.

## Signature

```powershell
_Get-FileSize [String]$Path
```

## Parameters

No parameters.

## Examples

No examples provided.

## Notes

This is an internal initialization function and should not be called directly.
