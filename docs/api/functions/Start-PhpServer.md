# Start-PhpServer

## Synopsis

Starts PHP built-in development server.

## Description

Wrapper for PHP built-in server command.

## Signature

```powershell
Start-PhpServer
```

## Parameters

### -Port

Port number for the server (default: 8000).


## Examples

### Example 1

`powershell
Start-PhpServer
``

### Example 2

`powershell
Start-PhpServer -Port 3000
``

## Aliases

This function has the following aliases:

- `php-server` - Starts PHP built-in development server.
