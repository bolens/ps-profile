# ConvertFrom-CsvToYaml

## Synopsis

Converts CSV file to YAML format.

## Description

Reads a CSV file and outputs its contents as YAML.

## Signature

```powershell
ConvertFrom-CsvToYaml
```

## Parameters

### -Path

The path to the CSV file to convert.


## Examples

### Example 1

```powershell
ConvertFrom-CsvToYaml -InputPath ./input.file
```

## Aliases

This function has the following aliases:

- `csv-to-yaml` - Converts CSV file to YAML format.
