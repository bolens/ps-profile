# Invoke-BunRun

## Synopsis

Runs npm scripts using Bun.

## Description

Wrapper for bun run command.

## Signature

```powershell
Invoke-BunRun
```

## Parameters

### -Arguments

Arguments to pass to bun run.


## Examples

### Example 1

`powershell
Invoke-BunRun build
``

### Example 2

`powershell
Invoke-BunRun dev
``

## Aliases

This function has the following aliases:

- `bun-run` - Runs npm scripts using Bun.
