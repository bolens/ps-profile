# Update-DenoSelf

## Synopsis

Updates Deno to the latest version.

## Description

Updates Deno itself to the latest version using 'deno upgrade'.

## Signature

```powershell
Update-DenoSelf
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Update-DenoSelf
    Updates Deno to the latest version.
``
