# Remove-FlutterPackage

## Synopsis

Removes packages from Flutter project.

## Description

Removes packages from pubspec.yaml.

## Signature

```powershell
Remove-FlutterPackage
```

## Parameters

### -Packages

Package names to remove.


## Examples

### Example 1

`powershell
Remove-FlutterPackage http
        Removes http from dependencies.
``

## Aliases

This function has the following aliases:

- `flutter-remove` - Removes packages from Flutter project.
