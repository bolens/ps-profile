# Get-WingetPackage

## Synopsis

Lists installed winget packages.

## Description

Shows all packages currently installed via winget.

## Signature

```powershell
Get-WingetPackage
```

## Parameters

### -Source

Filter by source.


## Examples

### Example 1

`powershell
Get-WingetPackage
        Lists all installed winget packages.
``

This function has the following aliases:

- `winget-list` - Lists installed winget packages.
