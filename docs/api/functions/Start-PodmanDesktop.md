# Start-PodmanDesktop

## Synopsis

Launches Podman Desktop GUI.

## Description

Starts Podman Desktop, a graphical interface for managing Podman containers, images, and volumes.

## Signature

```powershell
Start-PodmanDesktop
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Start-PodmanDesktop
        
        Launches Podman Desktop GUI.
``
