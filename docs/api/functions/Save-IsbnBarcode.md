# Save-IsbnBarcode

## Synopsis

Generates an EAN-13 barcode image for an ISBN.

## Description

Generates an EAN-13 barcode image for an ISBN.

## Signature

```powershell
Save-IsbnBarcode
```

## Parameters

### -Isbn

ISBN to encode as EAN-13.

### -OutputPath

Destination image path.

### -PassThru

Returns the output path.

System.String
