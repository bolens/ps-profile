# Get-EventSamplingStats

## Synopsis

Gets statistics about event sampling.

## Description

Returns statistics about how events are being sampled, useful for monitoring and adjusting sampling rates.

## Signature

```powershell
Get-EventSamplingStats
```

## Parameters

No parameters.

## Outputs

EventSamplingStats. Sampling statistics with type-safe properties and helper methods.


## Examples

### Example 1

```powershell
Get-EventSamplingStats
```

Returns sampling statistics.
