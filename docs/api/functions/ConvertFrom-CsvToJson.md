# ConvertFrom-CsvToJson

## Synopsis

Converts CSV file to JSON format.

## Description

Reads a CSV file and outputs its contents as JSON.

## Signature

```powershell
ConvertFrom-CsvToJson
```

## Parameters

### -Path

The path to the CSV file to convert.


## Examples

### Example 1

`powershell
ConvertFrom-CsvToJson -InputPath ./input.file
``

## Aliases

This function has the following aliases:

- `csv-to-json` - Converts CSV file to JSON format.
