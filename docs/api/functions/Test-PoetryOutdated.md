# Test-PoetryOutdated

## Synopsis

Checks for outdated Poetry dependencies.

## Description

Lists all packages that have newer versions available. This is equivalent to running 'poetry show --outdated'.

## Signature

```powershell
Test-PoetryOutdated
```

## Parameters

No parameters.

No examples provided.
