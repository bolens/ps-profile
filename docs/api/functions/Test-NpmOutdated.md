# Test-NpmOutdated

## Synopsis

Checks for outdated packages in the current project.

## Description

Lists all packages that have newer versions available. This is equivalent to running 'npm outdated'.

## Signature

```powershell
Test-NpmOutdated
```

## Parameters

No parameters.
