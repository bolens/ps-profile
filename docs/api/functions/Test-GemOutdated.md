# Test-GemOutdated

## Synopsis

Checks for outdated RubyGems packages.

## Description

Lists all installed gems that have newer versions available. This is equivalent to running 'gem outdated'.

## Signature

```powershell
Test-GemOutdated
```

## Parameters

No parameters.
