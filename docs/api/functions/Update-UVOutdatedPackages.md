# Update-UVOutdatedPackages

## Synopsis

Upgrades all outdated Python packages using uv.

## Description

Lists all outdated packages and upgrades them to their latest versions. This is a uv-compatible replacement for pip-based upgrade commands.

## Signature

```powershell
Update-UVOutdatedPackages
```

## Parameters

No parameters.

## Examples

### Example 1

`powershell
Update-UVOutdatedPackages
        Upgrades all outdated packages in the current environment.
``

## Aliases

This function has the following aliases:

- `uvupgrade` - Upgrades all outdated Python packages using uv.
