# _Get-HexDump

## Synopsis

Initializes file hex dump utility functions.

## Description

Sets up internal functions for hex dump operations. This function is called automatically by Ensure-FileUtilities.

## Signature

```powershell
_Get-HexDump [String]$Path
```

## Parameters

No parameters.

## Examples

No examples provided.

## Notes

This is an internal initialization function and should not be called directly.
