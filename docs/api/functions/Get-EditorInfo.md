# Get-EditorInfo

## Synopsis

Gets information about available editors.

## Description

Checks for installed editors and returns information about them. Lists available editors and their status.

## Signature

```powershell
Get-EditorInfo
```

## Parameters

No parameters.

## Outputs

System.Object[]. Array of editor information objects.


## Examples

### Example 1

`powershell
Get-EditorInfo
        
        Lists all available editors.
``
