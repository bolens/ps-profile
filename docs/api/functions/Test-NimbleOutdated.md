# Test-NimbleOutdated

## Synopsis

Checks for outdated Nim packages.

## Description

Lists all installed packages that have newer versions available. This is equivalent to running 'nimble outdated'.

## Signature

```powershell
Test-NimbleOutdated
```

## Parameters

No parameters.

No examples provided.
