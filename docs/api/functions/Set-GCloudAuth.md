# Set-GCloudAuth

## Synopsis

Manages Google Cloud authentication.

## Description

Wrapper for gcloud auth commands.

## Signature

```powershell
Set-GCloudAuth
```

## Parameters

### -Arguments

Arguments to pass to gcloud auth.


## Examples

### Example 1

`powershell
Set-GCloudAuth login
``

### Example 2

`powershell
Set-GCloudAuth list
``

## Aliases

This function has the following aliases:

- `gcloud-auth` - Manages Google Cloud authentication.
