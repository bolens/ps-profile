# Show-SystemDashboard

## Synopsis

Shows a comprehensive system status dashboard.

## Description

Displays CPU usage, memory usage, disk space, network status, and other system metrics in a clean, organized format.

## Signature

```powershell
Show-SystemDashboard
```

No parameters.
