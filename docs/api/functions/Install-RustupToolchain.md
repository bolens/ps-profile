# Install-RustupToolchain

## Synopsis

Installs Rust toolchains.

## Description

Wrapper for rustup install command.

## Signature

```powershell
Install-RustupToolchain
```

## Parameters

### -Arguments

Arguments to pass to rustup install.


## Examples

### Example 1

`powershell
Install-RustupToolchain stable
``

### Example 2

`powershell
Install-RustupToolchain nightly
``

## Aliases

This function has the following aliases:

- `rustup-install` - Installs Rust toolchains.

Defined in: ..\profile.d\rustup.ps1
