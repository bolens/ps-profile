# Test-CachedCommand

## Synopsis

Tests for command availability with caching.

## Description

Lightweight cached command testing used by profile fragments to avoid
        repeated Get-Command calls. Results are cached in script scope for performance.

## Signature

```powershell
Test-CachedCommand
```

No parameters.
