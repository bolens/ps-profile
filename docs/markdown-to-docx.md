# markdown-to-docx

## Synopsis

Converts Markdown file to DOCX.

## Description

Uses pandoc to convert a Markdown file to Microsoft Word DOCX format.

## Alias Information

**Alias for:** `ConvertTo-DocxFromMarkdown`

This is an alias that points to the `ConvertTo-DocxFromMarkdown` command. Use this alias as a shorthand for the full command name.
