# Test-IsMacOS

## Synopsis

Checks if the current platform is macOS.

## Description

Returns true if running on macOS, false otherwise.

## Signature

```powershell
Test-IsMacOS
```

## Parameters

No parameters.

## Outputs

System.Boolean. True if running on macOS, false otherwise.
