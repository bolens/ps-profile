# docx-to-latex

## Synopsis

Converts DOCX file to LaTeX.

## Description

Uses pandoc to convert a Microsoft Word DOCX file to LaTeX format.

## Alias Information

This is an alias that points to the `ConvertTo-LaTeXFromDocx` command. Use this alias as a shorthand for the full command name.
