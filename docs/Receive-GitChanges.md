# Receive-GitChanges

## Synopsis

Downloads objects and refs from remote repository.

## Description

Fetches changes from the remote repository without merging them into the current branch. Forwards all arguments to git fetch.

## Signature

```powershell
Receive-GitChanges
```

## Parameters

No parameters.
