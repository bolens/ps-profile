# Invoke-LastCommand

## Synopsis

Shows the last command matching a pattern.

## Description

Finds the most recent command matching the pattern and displays it for manual execution.

## Signature

```powershell
Invoke-LastCommand
```

## Parameters

### -Pattern

Pattern to match against command history.
